# Open Hearts Foundation — Child Sponsorship Platform

Child sponsorship platform connecting vulnerable children in Uganda with sponsors worldwide.

## Technology Stack

Next.js 15 (App Router), InsForge SDK (Supabase-compatible), Flutterwave (REST), Resend, Cloudflare Turnstile, Upstash Redis (REST), PostHog, Zod, Tailwind CSS v4, @base-ui/react, lucide-react.

## Getting Started

```bash
npm install
# Configure .env.local (see environment variables below)
npm run dev
```


### Build

```bash
npm run build
npm start
```
